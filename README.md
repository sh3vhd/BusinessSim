# 🏢 Бизнес Симулятор

**Онлайн-симулятор бизнеса** - это комплексная игра-стратегия, где вы создаёте и управляете виртуальной компанией, конкурируете с другими игроками и строите бизнес-империю.

## ✨ Особенности

### 🎯 Основные возможности

- **Создание компании** - Выберите одну из 6 отраслей и начните свой бизнес
- **Управление финансами** - Контролируйте доходы, расходы, кредиты и инвестиции
- **Продуктовая линейка** - Разрабатывайте и продвигайте продукты
- **Команда** - Нанимайте сотрудников, управляйте зарплатами и продуктивностью
- **Фондовый рынок** - Торгуйте акциями 5 различных секторов
- **Конкуренция** - Сражайтесь с 4 AI-конкурентами за долю рынка
- **Исследования** - Развивайте технологии для улучшения бизнеса
- **Строительство** - Возводите и улучшайте здания компании
- **Случайные события** - Экономические события влияют на игру
- **Достижения** - Разблокируйте 8 достижений

## 🚀 Быстрый старт

### Требования

- Node.js 18+ 
- npm или yarn

### Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/yourusername/business-simulator.git

# Перейдите в директорию проекта
cd business-simulator

# Установите зависимости
npm install

# Запустите в режиме разработки
npm run dev

# Соберите для продакшена
npm run build
```

### Локальный запуск

После установки проект будет доступен по адресу: `http://localhost:5173`

## 🎮 Как играть

### 1. Создание компании

1. Введите название вашей компании
2. Выберите отрасль
3. Подтвердите создание

### 2. Управление бизнесом

- **Обзор** - Следите за ключевыми показателями
- **Финансы** - Управляйте капиталом, берите кредиты, инвестируйте
- **Продукты** - Создавайте новые продукты, настраивайте цены
- **Сотрудники** - Нанимайте команду, следите за продуктивностью
- **Рынок** - Торгуйте акциями, следите за конкурентами
- **Исследования** - Развивайте технологии
- **Здания** - Стройте инфраструктуру

### 3. Скорость игры

- ⏸️ Пауза - Остановить время
- ▶️ 1x - Нормальная скорость
- ⏩ 2x - Ускоренная скорость
- ⏪ 5x - Максимальная скорость

## 📊 Игровая механика

### Финансовая система

```
Доходы = Продажи продуктов + Инвестиции
Расходы = Зарплаты + Содержание зданий + Кредиты + Маркетинг
Прибыль = Доходы - Расходы
```

### Прогрессия

- Опыт зарабатывается через выручку
- Уровень повышается каждые 1000 XP
- Репутация растёт с прибылью
- Новые возможности открываются с уровнем

### Случайные события

| Событие | Эффект | Длительность |
|---------|--------|--------------|
| Экономический бум | +20% спроса | 7 дней |
| Рецессия | -15% спроса | 10 дней |
| Техно-прорыв | +25% эффективности | 14 дней |
| Изменение налогов | -5% прибыли | 30 дней |
| Атака конкурента | -10% рынка | 5 дней |

## 🛠 Технологии

- **Frontend**: React 19.2.6, TypeScript 5.9.3
- **Styling**: TailwindCSS 4.1.17
- **Build Tool**: Vite 7.3.2
- **Charts**: Recharts
- **Icons**: Lucide React
- **State Management**: React Context + useReducer
